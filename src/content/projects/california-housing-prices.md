## Overview

A full-cycle machine learning project focused on predicting California housing prices from census data. The workflow spans EDA, feature engineering, model training, and deployment.

## Key Components

- Exploratory data analysis to identify statistical and geographical patterns.
- Feature engineering such as `rooms_per_household` and `bedrooms_per_room`.
- Regression modeling with Linear Regression, Random Forest, and XGBoost.
- Flask application deployment with containerization support.

## Results

| Model | MAE | RMSE | R2 Score |
|---|---:|---:|---:|
| Linear Regression | 48,660 | 67,305 | 0.67 |
| Random Forest | 32,187 | 49,243 | 0.82 |
| XGBoost | 31,827 | 48,125 | 0.83 |

## Impact

Demonstrates complete machine learning lifecycle capability from data preparation to production inference with measurable business insights.
