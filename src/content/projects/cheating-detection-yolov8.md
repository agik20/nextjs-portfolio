## Overview

A deep learning based real-time proctoring system that detects cheating activities during written exams. Built using YOLOv8 for object detection and pose estimation, the system identifies suspicious behaviors such as smartphone use and peer interaction.

## Highlights

- Designed for deployment on **Jetson Orin Nano** with 1080p camera input.
- Processes live video streams with low-latency inference.
- Delivers automated evidence capture with timestamps and classification labels.
- Assists academic integrity through intelligent monitoring automation.

## Impact

Helps institutions maintain fair and efficient exam supervision while proving that edge AI can scale for real-time operational environments.
