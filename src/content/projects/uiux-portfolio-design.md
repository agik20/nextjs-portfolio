## Overview

A personal portfolio website developed to present technical projects and skills in a modern, visually engaging interface. Built with **Next.js**, **TypeScript**, and **Tailwind CSS**, with focus on performance and maintainability.

## Highlights

- Integrated dark and light interaction patterns with smooth transitions.
- Structured content for fast rendering and SEO readiness.
- Responsive layout system for desktop, tablet, and mobile devices.
- Deployed on Vercel for reliable global delivery.

## Impact

Serves as a professional digital presence and demonstrates a scalable front-end architecture for continuous content growth.
