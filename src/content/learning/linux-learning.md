## Overview

Materi ini merangkum fondasi Linux dan system administration, mulai dari networking, DNS, NTP, keamanan, hingga package management untuk kebutuhan operasional server.

## Pokok Materi

- Konfigurasi interface jaringan, routing, dan konektivitas dasar.
- Implementasi DNS menggunakan BIND dan validasi query.
- Sinkronisasi waktu layanan dengan NTP untuk konsistensi sistem.
- Manajemen paket RPM/YUM untuk instalasi dan pemeliharaan software.
- Hardening server melalui firewall dan praktik keamanan dasar.

## Impact

Memberikan pemahaman operasional Linux yang aplikatif untuk membangun layanan yang stabil, aman, dan siap dikelola dalam skala produksi.
